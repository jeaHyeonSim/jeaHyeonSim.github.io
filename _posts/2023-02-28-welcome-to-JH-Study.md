## Contents
Jekyll은 또한 코드 스니펫에 대한 강력한 지원을 제공합니다:<br>
Jekyll also offers powerful support for code snippets:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
